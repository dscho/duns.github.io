Java

Since the release of Java in 1995 I have worked with the language. First on the WIRED project, later on in FreeHEP, Vector Graphics and the Maven-NAR-Plugin and more recently in projects such as OTP and ATWSS. I found it an easy language to learn, but it does take time to master all the libraries that come with it.

GWT

OTP first brought me into contact with AJAX and Google Web Toolkit (GWT). It compiles Java into JavaScript to run in a web browser. I later used GWT also on ATWSS and in SipML5. It has a lot of potential, as it is a lot easier (and safer) to write in Java than in JavaScript.
